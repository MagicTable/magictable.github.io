  • 网络通信是现代信息技术的基础。

简介

简要说明

  • 通过计算机网络将数据从发送方传输到接收方的过程。
  • 依赖于一系列协议（如 TCP/IP）来确保数据的正确传输。
  • 通常涉及物理层、数据链路层、网络层、传输层和应用层等多个网络层次。

主要功能

  • 数据传输：将数据从一个网络节点传输到另一个网络节点。
  • 数据交换：在网络中的多个节点之间交换数据。
  • 资源共享：允许网络中的用户共享硬件、软件和数据资源。
  • 通信服务：提供电子邮件、即时消息、文件传输和远程登录等服务。
  • 网络管理：监控网络状态，确保网络的稳定性和安全性。

注意事项

  • 安全性：确保数据传输的安全性，使用加密和认证机制来防止数据泄露和未授权访问。
  • 可靠性：选择合适的传输协议（如 TCP）来保证数据的可靠传输。
  • 性能：优化网络结构和协议配置，以提供高带宽和低延迟的通信服务。
  • 兼容性：确保网络设备和应用程序之间的兼容性，以便顺畅通信。
  • 可扩展性：设计网络时要考虑未来的扩展需求，以便于添加新设备和用户。

适用场景

  • 个人通信：如电子邮件、社交媒体、即时消息等。
  • 企业网络：用于企业内部的数据交换、资源共享和业务系统。
  • 互联网服务：提供网页浏览、在线视频、云服务等。
  • 远程工作：支持远程登录、虚拟私人网络（VPN）等远程办公解决方案。
  • 物联网（IoT）：连接各种设备和传感器，实现数据的收集和远程控制。
  • 移动通信：包括4G、5G等移动网络技术，适用于智能手机和其他移动设备。

网络通信三要素

[!info]

  • 协议：连接和数据在网络中传输的规则。
  • IP：设备在网络中唯一标识的地址。
  • 端口：应用程序在设备中唯一的标识。

IP

  • IP（Internet Protocol），全称“互联网协议地址”，是分配给联网设备在网络中唯一标识的地址。
  • IP地址有两种形式：IPv4、IPv6。
  • IPv4 由32位二进制组成，并将每8位二进制数转换为十进制数，使用点号.分开，例如：192.168.0.1
  • IPv6 由128位二进制组成，分成8段表示，每段每四位编码成一个十六进制位标识，数之间用冒号:分开，例如：2001:0db8:0000:2200:0008:0800:110a:417a

IP域名

  • 流程是输入域名发送到DNS服务器，查询域名对应的IP，再访问IP对应的服务器。例如：域名https://www.baidu.com，IP地址202.108.22.5

公网IP

  • 可以连接互联网的IP地址。

内网IP

  • 又称局域网IP，只能在同一网络下进行使用。
  • 路由以内的网络，可以连接互联网，但互联网无法直接连接内网（需要端口映射）。

特殊IP地址

  • 127.0.0.1、localhost：代表本机IP，只会寻找当前所在的主机。

端口

  • 标记正在计算机设备上运行的应用程序。
  • 被规定为一个16位的二进制数，范围是0~65535

分类

  • 周知（公认）端口：0~1023，即被预先定义的知名应用程序占用。（如：HTTP占用80、FTP占用21）
  • 注册端口：1024~49151，即分配给用户进程或某些应用程序。
  • 动态（私有）端口：49152~65535，动态端口一般分配给不固定分配某些进程。

提示

  • 设备中不能出现两个程序的端口号重复。
  • 个人开发者开发的程序一般选择使用注册端口。

协议

  • 网络上通信的设备，事先规定连接规则、传输数据的规则被称为网络通信协议。
  • OSI网络参考模型：全球网络互联标准。
  • TCP/IP网络模型：实际使用的国际标准。

  • 它定义了请求的格式、通信协议、数据交换方式以及响应的结构。

域名解析服务（DNS）

  • 域名解析服务（DNS）是互联网的一项基础服务。
  • 将易于记忆的域名转换成计算机能够直接访问的IP地址。
  • 这个过程允许用户通过输入简单的域名来访问互联网上的资源，而不需要记住复杂的数字序列。

网络域名备案（ICP）

  • 网络域名备案（ICP备案）是中国特有的互联网管理要求，全称为“互联网内容提供商备案”，是中国国家互联网信息办公室（原工业和信息化部）对在中国境内提供互联网信息服务的网站实行的登记制度。
