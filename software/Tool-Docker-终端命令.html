жӯӨйЎөеҶ…е®№
  • еҹәжң¬е‘Ҫд»Ө
  • й•ңеғҸе‘Ҫд»Ө
    • жҹҘжүҫй•ңеғҸпјҲиҝңзЁӢпјү
    • иҺ·еҸ–й•ңеғҸ

    • еҠ иҪҪй•ңеғҸ
  • е®№еҷЁе‘Ҫд»Ө
    • е‘Ҫд»ӨиЎҢжЁЎејҸиҝҗиЎҢе№¶иҝӣе…Ҙе®№еҷЁ
    • еҲӣе»әе№¶иҝҗиЎҢе®№еҷЁ

    • еҜјеҮәе®№еҷЁеҝ«з…§
    • еҜје…Ҙе®№еҷЁеҝ«з…§
  • cache
    • жҹҘиҜўй•ңеғҸдёӢиҪҪзҡ„дҝқеӯҳдҪҚзҪ®

ж–Үз« ж‘ҳиҰҒпјҡ ж‘ҳиҰҒеҶ…е®№гҖӮ

  • е‘Ҫд»ӨеҲҶдёәпјҡеҹәжң¬е‘Ҫд»ӨгҖҒй•ңеғҸе‘Ҫд»ӨгҖҒе®№еҷЁе‘Ҫд»ӨгҖҒд»“еә“е‘Ҫд»Ө

еҹәжң¬е‘Ҫд»Ө

жҹҘиҜўDockerзҡ„зүҲжң¬дҝЎжҒҜ

# зүҲжң¬еҸ·
docker -v

# зүҲжң¬дҝЎжҒҜ
docker version

docker info

её®еҠ©

# е‘Ҫд»ӨеҸҜйҖү
docker [е‘Ҫд»Ө] --help

й•ңеғҸе‘Ҫд»Ө

жҹҘжүҫй•ңеғҸпјҲиҝңзЁӢпјү

sudo docker search <й•ңеғҸеҗҚз§°>

иҺ·еҸ–й•ңеғҸ

# жӢүеҸ–й•ңеғҸ
sudo docker pull <й•ңеғҸеҗҚз§°>

# жҢҮе®ҡзүҲжң¬еҸ·
sudo docker pull <й•ңеғҸеҗҚз§°>:<зүҲжң¬еҸ·>

жҺЁйҖҒй•ңеғҸ

// дёҠдј жң¬ең°й•ңеғҸеҲ°е®ҳзҪ‘
sudo docker push <й•ңеғҸеҗҚз§°>

# жң¬ең°жһ„е»ә
sudo docker bulid

жҹҘзңӢй•ңеғҸпјҲжң¬ең°пјү

# еҲ—еҮәжң¬ең°дё»жңәзҡ„й•ңеғҸ
sudo docker images
