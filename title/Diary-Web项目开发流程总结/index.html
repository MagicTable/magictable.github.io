IDEA项目创建流程

  1. 使用 IDEA 创建一个新项目

    • 构建系统选择：Maven

  2. 将Maven添加到IDEA中

    • 点击菜单栏打开“设置”窗口：文件 -> 设置

    • 切换到“构建、执行、部署 -> 构建工具 -> Maven”页面，修改“Maven主路径、用户配置文件、本地仓库”的路径。

    • 点击应用，并点击确定

  3. 将Tomcat添加到IDEA中

    • 点击菜单栏打开“设置”窗口：文件 -> 设置

    • 切换到“构建、执行、部署 -> 应用程序服务器”页面，点击“+”添加“Tomcat服务器”

    • 填写“Tomcat的所在目录”

    • 点击应用，并点击确定

  4. 修改pom.xml配置文件，创建Web项目

    提示：该步骤将会自动为当前模块创建“Web模块”和两个“工件”。

    • 在文件中添加<packaging>war</packaging>，并点击刷新按钮。
    • 添加图片中的第11行的代码。

  5. 为当前模块创建“Web模块”

    • 若“Web项目”已存在则无需再次创建。
    • 请确认“部署描述符”是否已添加，若没有则需要点击“+”创建，目录结构是WEB-INF/web.xml，该文件是Web服务器的配置文件。（创建位置无要求）
    • 请确认“Web资源目录”是否已添加，若没有则需要点击“+”创建，默认位置是src/main/webapp，该文件是存放Web资源的目录。

  6. 为当前模块创建“工件”

    • 若“工件”已存在则无需再次创建。

    • 提示：归档型（将文件打包压缩成一个.war文件，方便部署到生产环境），展开型（保留项目目录结构没有压缩，用于测试开发方面修改文件）

  7. 添加运行/调试配置

    作用：提供一键式在Tomcat服务器上运行Web应用。

    • 点击菜单栏打开“运行/调试窗口”窗口：运行 -> 编辑配置
    • 点击“+”添加“Tomcat服务器 -> 本地”
    • 点击切换到“部署”页面，点击“+”添加“工件”
    • 点击应用，并点击确定
