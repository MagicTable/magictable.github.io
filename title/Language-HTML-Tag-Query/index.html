表格

<table>支持的属性

  • border:<数值>：表格线框宽度。
  • <th> </th>：列标题
  • <tr> </tr>：行标签
  • <tb> </tb>：列标签

超链接

<a>支持的属性

href:<网址>

  • 要链接的网址。

target:<打开方式>

  • 要如何打开链接的页面。
  • _blank新页面打开、_parent当前页面跳出父框架打开、_self当前页面打开、_top当前页面最外层框架打开。

内联框架

<iframe>支持的属性

src=<网址>

  • 指向显示的位置。

width

  • 设置宽度

height

  • 设置高度

frameborder

  • 边框宽度

图片

<img />支持的属性

src:<图片地址>

  • 本地文件目录、网络地址。

alt:<文本>

  • 若图片未加载时，显示文本。

width:<数值>

  • 图片宽度。

height:<数值>

  • 图片高度。
