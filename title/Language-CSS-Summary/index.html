简介

  • CSS层叠样式表（Cascading Style Sheets）。
  • 是一种用来为结构化文档添加样式（字体、间距和颜色等）的计算机语言。
  • 样式定义如何显示HTML元素。
  • 使多个样式可以层叠到一个元素。
  • 可以将内容和表现分离，分工明确提高开发效率。
  • 外部样式表通常存储在css文件中。
  • CSS文件的扩展名为.css。

语法格式

  • CSS每个语句结束后需要有;分号结尾。

注释

CSS样式引入方式

行内样式

  • 在标签中写style属性，属性中写样式参数。

内联样式

  • 当单个文档需要特殊的样式时，就应该使用内联样式表。你可以使用<style> </style>标签在文档<head></head>头部定义内部样式表。

外联样式

  • 将样式放在外部，内容与样式分离。

样式优先级

  • 就近原则
  • 行内 > 内联 > 外联

选择器

元素（标签）选择器

  • 选择器的名字必须是标签的名字。
  • 作用是选择器中的样式会作用于所有相同标签上。

id选择器

  • id 选择器可以为标有特定 id 的 HTML 元素指定特定的样式。
  • HTML元素以id属性来设置id选择器。
  • CSS 中 id 选择器以#井号来表示。
  • id名称不能重复，即不能重复在不同的元素中使用同一个id名称。

class选择器

  • class 选择器用于描述一组元素的样式，class 选择器有别于id选择器，class可以在多个元素中使用。
  • class 选择器在 HTML 中以 class 属性表示。
  • 在 CSS 中，class选择器以一个.点号表示。

CSS样式属性和参数

文字属性

设置文本颜色

属性名

  • color

参数值

  • 十六进制 ：例如#ff0000
  • RGB ：例如rgb(255,0,0)
  • 颜色名称：例如red红色、gray灰色、black黑色、white白色、purple紫色

案例

设置文本对齐方式

属性名

  • direction

参数值

  • ltr：（默认）从左往右。
  • rtl：从右往左。

案例

设置字符间距

属性名

  • letter-spacing

参数值

  • normal：（默认）规定字符间没有额外的空间。
  • 数值px：指定字符间的固定间距，允许为负值。

背景样式

设置背景颜色

属性名

  • background-color

参数值

  • 十六进制 ：例如#ff0000
  • RGB ：例如rgb(255,0,0)
  • 颜色名称：例如red红色、gray灰色、black黑色、white白色、purple紫色

案例

将图像设置为背景

属性名

  • background-image

参数值

  • url(<图片文件>)：本地文件路径/网络URL地址。

案例

设置图片平铺方式

属性名

  • background-repeat

参数值

  • repeat-x：水平方向平铺。
  • repeat-y：垂直方向平铺。
  • no-repeat：不进行平铺。

案例

设置图片定位方式

属性名

  • background-position

参数值

  • center：居中。
  • left：左边。
  • right：右边。
  • top：顶部。
  • bottom：底部。
  • x% y%：0%代表左，50%代表居中，100%代表右。

案例

设置背景图片是否固定

属性名

  • background-attachment

参数值

  • scroll：（默认）背景图片随着页面的滚动而滚动。
  • fixed：背景图片不会随着页面的滚动而滚动。
  • local：背景图片会随着元素内容的滚动而滚动。
