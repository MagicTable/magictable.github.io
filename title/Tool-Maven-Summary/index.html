摘要： Maven 是一个Java项目管理和构建工具。

简介

Maven 是一个Java项目管理和构建工具。

  • 提供一套标准化的项目结构。
  • 并使用统一的方式进行自动化构建。
  • 管理项目依赖的jar包。

清理、编译、测试、报告、打包、部署

jar 包之间的依赖类似于面向对象编程中的继承，且是同时支持多个继承的结构。

安装

本地仓库存储位置

  • Debian Linux 系统默认下载到本地仓库的保存位置在：/home/<用户名>/.m2/repository/
  • Windows 系统默认下载到本地仓库的保存位置在：C:/<用户名>/.m2/repository/

修改镜像源
