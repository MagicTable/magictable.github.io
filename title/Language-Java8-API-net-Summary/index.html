本机IP地址

  • 包路径：java.net.InetAddress

构造方法

方法 说明
InetAddress.getLocalHost() 获取本机IP，返回inetAddress对象
InetAddress.getLocalHost().getHostName() 获取给ip地址对象对应的主机名
InetAddress.getLocalHost().getHostAddress() 获取该ip地址对象中的ip地址信息
InetAddress.getByName(String host) 根据ip地址或域名，返回inetAddress对象
InetAddress.getByName(String host).isReachable(int timeout) 在指定的毫秒时间内判断主机与该ip对应的主机是否能连通
