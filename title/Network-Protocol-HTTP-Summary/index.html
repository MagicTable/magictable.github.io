摘要： 在网络模型中属于应用层的网络通讯协议，HTTP全称：超文本传输协议。

简介

  • URI：统一资源标识符，代表网络中某个资源的名称。
  • URL：统一资源定位符，网址，代表网络设备的地址。
  • URN：

例子：http://www.baidu.com:80/index.html?name=xiaoming&age=18#a

  • 协议：http
  • IP/域名：www.baidu.com
  • 端口：80
  • 请求资源：/index.html
  • 参数：name=xiaoming&age=18
  • 锚点：a

请求格式

  • 请求头中的内容是以“键值对”的形式存储。
