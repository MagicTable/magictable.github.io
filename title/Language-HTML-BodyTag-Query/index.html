class属性

对 HTML 进行分类（设置类），使我们能够为元素的类定义 CSS 样式。
为相同的类设置相同的样式，或者为不同的类设置不同的样式。
一个元素能指定多个类。

id属性

定义元素唯一的id。
id 属性指定 HTML 元素的唯一 ID。 id 属性的值在 HTML 文档中必须是唯一的。
id 属性用于指向样式表中的特定样式声明。
JavaScript 也可使用它来访问和操作拥有特定 ID 的元素。

语法规则

写一个井号 #，后跟一个 id 名称。然后，在花括号 {} 中定义 CSS 属性。

style属性

规定元素的行为样式。

支持参数

background-color:<颜色>

  • 定义背景颜色。

font-family:<字体名>

  • 定义字体系列。

color:<颜色名>

  • 定义字体颜色。

font-size:<数值>px

  • 定义字体尺寸，格式：<数值>px

text-align:<对齐方式>

  • 文本的水平对齐方式
  • 支持参数：center居中对齐，…

表格

介绍

  • <table> </table>：表格
  • <thead> </thead>：标头
  • <tbody> </tbody>：表体
  • <tfoot> </tfoot>：表尾
  • <tr> </tr>：行
  • <th> </th>：列（居中+加粗）
  • <tb> </tb>：列（无）

属性

  • border

  • 功能：表格线框宽度。

  • 参数：数值。

  • colspan

  • 功能：向右合并，跨列合并。

  • 参数：整数。

  • rowspan

  • 功能：向下合并，跨行合并。

  • 参数：整数。

  • 功能：指定资源路径。

  • 功能：为资源添加控制器。

超链接

介绍

  • <a> </a>：通过网址或地址跳转到另一个页面。

属性

href

  • 功能：要链接的网址。
  • 参数：字符串类型的网址。

  • 功能：要如何打开链接的页面。
