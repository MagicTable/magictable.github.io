摘要： 关键字是Java编程语言中的语法规则语句，具有特殊含义，用于定义程序的基本结构和元素。

  • 关键字是固定的，不能更改或重新定义。

访问权限关键字

标识符 说明
private 私有的，本类可见
protected 受保护的，本类、子类、同一包中的类可见
default 默认（可不写），本类、同一包中的类可见
public 公共的，对任何类可见

类、抽象、接口关键字

标识符 说明
class 定义类
abstract 定义抽象类
extends 类继承抽象类/实现类
interface 定义接口类
implements 类继承接口类

数据类型关键字

标识符 说明
byte 字节型
short 短整型
int 整型
long 长整型
float 单精度浮点
double 双精度浮点
char 字符型
boolean 布尔型
enum 枚举类型

流程控制关键字

标识符 说明
if 如果
else 否则
switch 根据值选择执行
case 定义一个值以供switch选择
default 在switch语句中定义默认分支
while 循环
do 运行
for 循环
break 跳出本层整个循环
continue 跳出本层当前循环
return 返回

异常处理关键字

标识符 说明
try 抛出异常
catch 捕获异常
finally 有没有异常都执行，垃圾回收器
throw 手动抛出异常对象
throws 声明一个方法中可能会抛出异常

非访问权限（声明）

标识符 说明
final 最终，不可改变的
static 声明静态

其他关键字

标识符 说明
package 包
import 引入
void 无返回值
new 实例化对象
this 当前对象，调用构造方法中的变量
super 引用基类（父类、超类）
instanceof 判断某一个对象的类型是否继承自另一个类型中
